# 列表

## range

* 作用：生成区间

* 参数作用：区间结束值

* 示例：(range 3)

## list

* 参数作用：元素<\*>

## \+\+

* 作用：拼接列表（列表与列表或列表与元素）
* 参数数量：无穷（>= 2）
* 参数作用：列表|元素
* 参数类型：list|token
* 返回值：拼接的列表
* 返回类型：list
* 示例：(\+\+ (list 2 1 3) (list 3 4 7))

## length

* 作用：获取列表长度
* 参数数量：1
* 参数作用：列表
* 参数类型：list
* 返回值：长度
* 返回类型：int
* 示例：(length (list 1 2 3))

## map

* 作用：将列表应用于procedure并返回运算结果构成的列表
* 参数数量：无穷（>= 2）
* 参数作用：过程、列表<\*>
* 参数类型：procedure、list<\*>
* 返回值：运算结果构成的列表
* 返回类型：list
* 示例：(map + (list 1 2) (list 4 5))

## filter

* 作用：将列表应用与procedure并返回符合条件的结果构成的列表
* 参数数量：2
* 参数作用：过程、列表
* 参数类型：procedure、list
* 返回值：符合条件的结果构成的列表
* 返回类型：list
* 示例：(filter + (lambda (x) (> x 2)) (list 1 2 3 4))

## for-each

* 作用：将列表应用于procedure
* 参数数量：无穷（>= 2）
* 参数作用：过程、列表<\*>
* 参数类型：procedure、list<\*>
* 返回值：nil
* 返回类型：word
* 示例：(for-each (lambda (x y) (println (+ x y))) (list 1 2) (list 4 5))

## foldr

* 作用：从右向左依次将列表元素与累积值应用于过程，并最终返回一个单一结果
* 参数数量：3
* 参数作用：过程、补充值、列表
* 参数类型：procedure、token、list
* 返回值：对列表折叠后的最终结果
* 返回类型：token
* 示例：(foldr + 0 (list 1 2 3))

## foldl
