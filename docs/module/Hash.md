# 哈希

需要导入模块：hash

## md5

* 作用：MD5
* 参数数量：1
* 参数作用：\[string|binary\]
* 参数类型：\[string|list\]
* 返回值：MD5
* 返回类型：string
* 示例：(md5 "abc")

## md5

* 作用：MD5 (采用特定编码格式）
* 参数数量：1
* 参数作用：string、编码格式
* 参数类型：string、string
* 返回值：MD5
* 返回类型：string
* 示例：(md5 "abc" "UTF-8")
